# Data-Engineering- — Building Data-Driven Solutions with Python

## Overview
This repository contains a collection of projects developed during the Lux Dev 6-month bootcamp, focusing on data engineering with Python. The projects showcase various data manipulation, analysis, and visualization techniques using popular libraries and tools in the Python ecosystem.

## Tech Stack
* Python 3.12
* pip 24.0
* Jupyter Notebook (via main.ipynb)
* CSV and JSON data formats (via MOCK_DATA.csv and MOCK_DATA.json)

## Features
* Data ingestion and processing using Python's built-in libraries
* Data visualization using Jupyter Notebook's interactive features
* Environment variable management using virtual environments (via sample/.env)
* pip package management for dependency installation and updates

## Setup & Installation

1. Clone the repository using Git:
```bash
git clone https://github.com/abdiomari/Data-Engineering-.git
```
2. Navigate to the project directory:
```bash
cd Data-Engineering-
```
3. Activate the virtual environment (for Windows):
```bash
sample\.env\bin\Activate.ps1
```
4. Install dependencies using pip:
```bash
pip install -r requirements.txt
```
5. Run the Jupyter Notebook:
```bash
jupyter notebook
```
6. Open a web browser and navigate to `http://localhost:8888` to access the Jupyter Notebook interface.

Note: This setup assumes a Windows environment. For other operating systems, please adjust the commands accordingly.
