# Data-Engineering- — Building Data-Driven Solutions

## Overview
This repository contains a collection of projects developed during the Lux Dev 6-month bootcamp, focusing on data engineering. The projects showcase various data manipulation, analysis, and visualization techniques using popular libraries and tools Data Analysis and Engineering ecosystem.

## Tech Stack and Tools
* Python
* Jupyter Notebook
* Ms Power BI
* Azure
* Apache Airflow
* Apache Kafka
* PySpark

## Features
* Data ingestion and processing using Python's built-in libraries
* Data visualization using MS Power BI
